A New Blog Move

Jan 25th, 2014

Moving

I’ve decided to move my blog to a jekyll-powered github pages blog.

It’s not that I didn’t like Google’s blogger, but there simply existed no easy-to-use syntax highlighter utilities.

Most folks I know on google’s blogger service used “Syntax Highlighter” by Alex Gorbatchev. To be fair Gorbatchev’s stylesheet is quite nice — beautiful even — but implementing it in Blogger was a pain. I had to essentially edit the post in HTML mode to keep it from going crazy with the formatting.

Besides, using github is a more natural workflow for those that are used to it. Writing blog posts in vim also feels nice.

One thing that I’ll miss is Google’s fantastic SEO optimization. Oh well. Some things are worth it.
